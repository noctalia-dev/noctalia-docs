Take full control of Noctalia with keyboard shortcuts and IPC (Inter-Process Communication). This page covers all available commands you can bind to your favorite keys.

<Aside type="tip" title="Quick Shell Command Prefix">
Depending on your installation, the command prefix may vary:
- **NixOS Flake**: Use `noctalia-shell` directly.
- **PikaOS / Others**: Use `qs -c noctalia-shell`.
- **Manual**: Use `qs` if you have Noctalia in `~/.config/quickshell/`.

*In the tables below, we use `qs -c noctalia-shell` as the default.*
</Aside>

## Available Commands

### Core Functions

| Function | Command | Description |
|----------|---------|-------------|
| **Launcher** | `ipc call launcher toggle` | Open/close the application launcher |
| **Control Center** | `ipc call controlCenter toggle` | Toggle the control center visibility |
| **Settings** | `ipc call settings toggle` | Open/close the settings window |

### Quick Access

| Function | Command | Description |
|----------|---------|-------------|
| **Clipboard** | `ipc call launcher clipboard` | Access your clipboard history |
| **Emoji** | `ipc call launcher emoji` | Quick emoji selector access |
| **Command** | `ipc call launcher command` | Quick command runner access |
| **Calendar** | `ipc call calendar toggle` | Open/close the Calendar window |
| **Session** | `ipc call sessionMenu toggle` | Logout, reboot, shutdown... |
| **Lock & Suspend** | `ipc call sessionMenu lockAndSuspend` | Lock screen and suspend system |
| **System Monitor** | `ipc call systemMonitor toggle` | Open/close system monitor |

### Feedback & UI

#### Toast Notifications

Display transient messages with a JSON payload.

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `title` | string | Yes | Toast title |
| `body` | string | No | Toast body/description |
| `type` | string | No | `notice` (default), `warning`, or `error` |

**Example:**
```bash
qs -c noctalia-shell ipc call toast send '{"title": "Low battery", "type": "warning"}'
```

| Function | Command | Description |
|----------|---------|-------------|
| **Settings** | `ipc call plugin openSettings $pluginId` | Open plugin settings |

## Example Configurations

Apply these binds in your window manager configuration to integrate Noctalia.

<Tabs>
  <TabItem label="Niri">
    <Steps>
    1. Open `~/.config/niri/config.kdl`.
    2. Add the following to your `binds` section:

        ```kdl
        // Core Noctalia binds
        Mod+Space { spawn "qs" "-c" "noctalia-shell" "ipc" "call" "launcher" "toggle"; }
        Mod+S { spawn "qs" "-c" "noctalia-shell" "ipc" "call" "controlCenter" "toggle"; }
        Mod+Comma { spawn "qs" "-c" "noctalia-shell" "ipc" "call" "settings" "toggle"; }

        // Audio & Brightness
        XF86AudioRaiseVolume { spawn "qs" "-c" "noctalia-shell" "ipc" "call" "volume" "increase"; }
        XF86AudioLowerVolume { spawn "qs" "-c" "noctalia-shell" "ipc" "call" "volume" "decrease"; }
        XF86AudioMute { spawn "qs" "-c" "noctalia-shell" "ipc" "call" "volume" "muteOutput"; }
        XF86MonBrightnessUp { spawn "qs" "-c" "noctalia-shell" "ipc" "call" "brightness" "increase"; }
        XF86MonBrightnessDown { spawn "qs" "-c" "noctalia-shell" "ipc" "call" "brightness" "decrease"; }
        ```
    </Steps>
  </TabItem>

  <TabItem label="Hyprland">
    <Steps>
    1. Open `~/.config/hypr/hyprland.conf`.
    2. Add these binds:

        ```ini
        $ipc = qs -c noctalia-shell ipc call

        # Core binds
        bind = SUPER, SPACE, exec, $ipc launcher toggle
        bind = SUPER, S, exec, $ipc controlCenter toggle
        bind = SUPER, comma, exec, $ipc settings toggle

        # Media keys
        bindel = , XF86AudioRaiseVolume, exec, $ipc volume increase
        bindel = , XF86AudioLowerVolume, exec, $ipc volume decrease
        bindl = , XF86AudioMute, exec, $ipc volume muteOutput
        bindel = , XF86MonBrightnessUp, exec, $ipc brightness increase
        bindel = , XF86MonBrightnessDown, exec, $ipc brightness decrease
        ```
    </Steps>
  </TabItem>
</Tabs>

## Discovering Commands

Not sure what else you can control? Noctalia allows you to inspect available IPC targets and handlers directly from the terminal.

**List all targets**:
    ```bash
    qs -c noctalia-shell ipc show
    ```

### Commands with Arguments

Some handlers require arguments. Provide them sequentially after the command name.

```bash
# Set specific wallpaper
qs -c noctalia-shell ipc call wallpaper set "/path/to/image.jpg" "DP-1"

# Seek media relative (5 seconds)
qs -c noctalia-shell ipc call media seekRelative 5
```

<Aside type="tip" title="Custom Shell Scripts">
Combine multiple IPC calls for complex automated workflows.

```bash
#!/bin/bash
# High Performance Mode + Dark Theme
qs -c noctalia-shell ipc call powerProfile set performance
qs -c noctalia-shell ipc call darkMode setDark
```
