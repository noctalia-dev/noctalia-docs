Manage your hardware and system state directly through the shell.

### Audio Management

| Function | Command | Description |
|----------|---------|-------------|
| **Volume Up** | <IpcCommand prefix="ipc call" command="volume increase" /> | Increase output volume |
| **Volume Down** | <IpcCommand prefix="ipc call" command="volume decrease" /> | Decrease output volume |
| **Mute Output** | <IpcCommand prefix="ipc call" command="volume muteOutput" /> | Toggle output mute |
| **Input Up** | <IpcCommand prefix="ipc call" command="volume increaseInput" /> | Increase input volume |
| **Input Down** | <IpcCommand prefix="ipc call" command="volume decreaseInput" /> | Decrease input volume |
| **Mute Input** | <IpcCommand prefix="ipc call" command="volume muteInput" /> | Toggle input mute |
| **Toggle Panel** | <IpcCommand prefix="ipc call" command="volume togglePanel" /> | Toggle audio panel |
| **Open Panel** | <IpcCommand prefix="ipc call" command="volume openPanel" /> | Open audio panel |
| **Close Panel** | <IpcCommand prefix="ipc call" command="volume closePanel" /> | Close audio panel |

### Media Controls

| Function | Command | Description |
|---|---|---|
| **Play/Pause** | <IpcCommand prefix="ipc call" command="media playPause" /> | Toggle playback |
| **Play** | <IpcCommand prefix="ipc call" command="media play" /> | Play media |
| **Pause** | <IpcCommand prefix="ipc call" command="media pause" /> | Pause media |
| **Next** | <IpcCommand prefix="ipc call" command="media next" /> | Next track |
| **Previous** | <IpcCommand prefix="ipc call" command="media previous" /> | Previous track |
| **Seek Rel.** | <IpcCommand prefix="ipc call" command="media seekRelative $offset" /> | Seek by seconds |
| **Seek By Ratio** | <IpcCommand prefix="ipc call" command="media seekByRatio $position" /> | Seek to position (0.0-1.0) |
| **Toggle Panel** | <IpcCommand prefix="ipc call" command="media toggle" /> | Toggle media panel |
