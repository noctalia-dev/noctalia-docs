### Toast Notifications

| Function | Command | Description |
|----------|---------|-------------|
| **Send** | <IpcCommand prefix="ipc call" command="toast send '$json'" /> | Display a toast notification |
| **Dismiss** | <IpcCommand prefix="ipc call" command="toast dismiss" /> | Dismiss active toast notification |

The toast `send` command accepts a JSON object with the following fields:

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `title` | string | Yes | Toast title |
| `body` | string | No | Toast body/description |
| `icon` | string | No | Icon name (only for notice type) |
| `type` | string | No | `notice` (default), `warning`, or `error` |
| `duration` | number | No | Duration in ms (defaults: notice=3000, warning=4000, error=6000) |

**Examples:**
```bash
# Simple notice
qs -c noctalia-shell ipc call toast send '{"title": "Recording", "body": "Started (AV1 HDR)", "icon": "media-record"}'

# Warning toast
qs -c noctalia-shell ipc call toast send '{"title": "Low battery", "type": "warning"}'

# Error with custom duration
qs -c noctalia-shell ipc call toast send '{"title": "Connection failed", "body": "Check your network", "type": "error", "duration": 10000}'
```
### Appearance

#### Visibility

| Function | Command | Description |
|----------|---------|-------------|
| **Toggle Bar** | <IpcCommand prefix="ipc call" command="bar toggle" /> | Toggle bar visibility |
| **Show Bar** | <IpcCommand prefix="ipc call" command="bar showBar" /> | Show bar |
| **Hide Bar** | <IpcCommand prefix="ipc call" command="bar hideBar" /> | Hide bar |
| **Peek Bar** | <IpcCommand prefix="ipc call" command="bar peek" /> | Briefly show bar, then auto-hide |
| **Bar DisplayMode** | <IpcCommand prefix="ipc call" command="bar setDisplayMode <mode> <screen>" /> | Set bar display mode |
| **Bar Position** | <IpcCommand prefix="ipc call" command="bar setPosition <position> <screen>" /> | Set bar position |
| **Toggle Dock** | <IpcCommand prefix="ipc call" command="dock toggle" /> | Toggle dock visibility |
| **Show Widgets** | <IpcCommand prefix="ipc call" command="desktopWidgets enable" /> | Enable desktop widgets |
| **Hide Widgets** | <IpcCommand prefix="ipc call" command="desktopWidgets disable" /> | Disable desktop widgets |
| **Toggle Widgets** | <IpcCommand prefix="ipc call" command="desktopWidgets toggle" /> | Toggle desktop widgets |
| **Edit Widgets** | <IpcCommand prefix="ipc call" command="desktopWidgets edit" /> | Toggle widget editing |

:::note[Available Bar Display Modes]
- `always_visible`
- `non_exclusive`
- `auto_hide`

`<screen>`: Monitor identifier (e.g., `HDMI-A-1`), or use an empty string or `all` to set mode on all monitors.
:::

:::note[Available Bar Positions]
- `top`
- `bottom`
- `left`
- `right`

`<screen>`: Monitor identifier (e.g., `HDMI-A-1`). Use an empty string or `all` to set position on all monitors.
:::

#### Wallpaper Management

| Function | Command | Description |
|----------|---------|-------------|
| **Selector** | <IpcCommand prefix="ipc call" command="wallpaper toggle" /> | Toggle wallpaper selector |
| **Get Wallpaper** | <IpcCommand prefix="ipc call" command="wallpaper get $monitor" /> | Get path to  wallpaper on monitor |
| **Set Wallpaper** | <IpcCommand prefix="ipc call" command="wallpaper set $path $monitor" /> | Set wallpaper on monitor |
| **Random** | <IpcCommand prefix="ipc call" command="wallpaper random $monitor" /> | Apply random wallpaper (omit monitor for all) |
| **Toggle Auto** | <IpcCommand prefix="ipc call" command="wallpaper toggleAutomation" /> | Toggle automation |
| **Enable Auto** | <IpcCommand prefix="ipc call" command="wallpaper enableAutomation" /> | Enable automation |
| **Disable Auto** | <IpcCommand prefix="ipc call" command="wallpaper disableAutomation" /> | Disable automation |
| **Refresh List** | <IpcCommand prefix="ipc call" command="wallpaper refresh" /> | Refresh wallpapers list |

:::note[Wallpaper variables]
- `$path`: Full path to your wallpaper image file
- `$monitor`: Monitor identifier (e.g., `HDMI-A-1`). Use an empty string or `all` to apply to all monitors.
:::
