These commands open, close, or adjust Noctalia UI surfaces.

## Bar

| Action | Command | Description |
|--------|---------|-------------|
| Show bar | <IpcCommand command="bar-show" /> [bar-name] [monitor-selector] | Reveal matching bar instance(s). Omit both arguments to show every bar on every output. |
| Hide bar | <IpcCommand command="bar-hide" /> [bar-name] [monitor-selector] | Hide matching bar instance(s) and block edge/pointer reveal until the next show/toggle. Omit both arguments to hide every bar. |
| Toggle bar | <IpcCommand command="bar-toggle" /> [bar-name] [monitor-selector] | Toggle visibility for matching bar instance(s). Omit both arguments to toggle every bar. Does not block edge reveal on hide. |
| Set bar auto-hide | <IpcCommand command="bar-auto-hide-set" /> &lt;on\|off&gt; [bar-name] [monitor-selector] | Temporarily enable or disable auto-hide for matching bar instance(s). Omit `bar-name` to update every bar. |
| Set bar layer | <IpcCommand command="bar-layer-set" /> &lt;top\|overlay&gt; [bar-name] [monitor-selector] | Temporarily move matching bar instance(s) to the `top` or `overlay` layer-shell layer. `overlay` shows the bar above fullscreen apps; attached panels follow the bar's layer. Omit `bar-name` to update every bar. |

Use <IpcCommand command="bar-hide" /> when you want the bar to stay hidden until <IpcCommand command="bar-show" />.

## Panels

| Action | Command | Description |
|--------|---------|-------------|
| Open panel | <IpcCommand command="panel-open <id> [context]" /> | Open a panel by id without toggling it closed when it is already open. Optional `context` works like <IpcCommand command="panel-toggle <id> [context]" />. |
| Close panel | <IpcCommand command="panel-close [id]" /> | Close the active panel, or close the named panel if it is active. |
| Toggle launcher | <IpcCommand command="panel-toggle launcher [query]" /> | Open or close the app launcher. Optional `query` pre-fills the search input (e.g. `noctalia msg panel-toggle launcher "/wall"`). |
| Toggle session menu | <IpcCommand command="panel-toggle session" /> | Open or close the logout, reboot, and shutdown menu. |
| Toggle clipboard | <IpcCommand command="panel-toggle clipboard" /> | Open or close clipboard history. |
| Toggle wallpaper panel | <IpcCommand command="panel-toggle wallpaper" /> | Open or close wallpaper picker. |
| Toggle control center | <IpcCommand command="panel-toggle control-center [tab]" /> | Open or close Control Center. Optional `tab` opens the tab inside control center by default (e.g. `noctalia msg panel-toggle control-center media`). |
| Toggle plugin panel | <IpcCommand command="panel-toggle <author/plugin:entry> [context]" /> | Open or close a plugin `[[panel]]` entry (e.g. `noctalia msg panel-toggle noctalia/wallhaven:browser`). Optional `context` is passed to `onOpen`. |

## Dock

| Action | Command | Description |
|--------|---------|-------------|
| Show dock | <IpcCommand command="dock-show" /> | Show all dock instances and save that override. |
| Hide dock | <IpcCommand command="dock-hide" /> | Hide all dock instances and save that override. |
| Toggle dock | <IpcCommand command="dock-toggle" /> | Toggle dock visibility and save the new state. |
| Reload dock | <IpcCommand command="dock-reload" /> | Reload dock configuration. |

## Desktop Widgets

| Action | Command | Description |
|--------|---------|-------------|
| Enter edit mode | <IpcCommand command="desktop-widgets-edit" /> | Start moving, resizing, rotating, and removing desktop widgets. |
| Exit edit mode | <IpcCommand command="desktop-widgets-exit" /> | Leave desktop widget edit mode. |
| Toggle edit mode | <IpcCommand command="desktop-widgets-toggle-edit" /> | Switch edit mode on or off. |
| Show widgets | <IpcCommand command="desktop-widgets-show" /> | Reveal desktop widgets now, even when the saved setting is disabled. Runtime only - does not change the saved setting, and resets on restart. |
| Hide widgets | <IpcCommand command="desktop-widgets-hide" /> | Hide desktop widgets now, tearing down their instances so rendering and compute stop. Runtime only - does not change the saved setting, and resets on restart. |
| Toggle widgets | <IpcCommand command="desktop-widgets-toggle" /> | Flip current visibility; prints `shown` or `hidden`. Runtime only - does not change the saved setting. Changing the saved Desktop > Widgets toggle cancels the override. |

## Lockscreen Widgets

| Action | Command | Description |
|--------|---------|-------------|
| Enter edit mode | <IpcCommand command="lockscreen-widgets-edit" /> | Edit lockscreen widget layout (session must be unlocked). |
| Exit edit mode | <IpcCommand command="lockscreen-widgets-exit" /> | Leave lockscreen widget edit mode. |
| Toggle edit mode | <IpcCommand command="lockscreen-widgets-toggle-edit" /> | Switch lockscreen widget edit mode on or off. |
