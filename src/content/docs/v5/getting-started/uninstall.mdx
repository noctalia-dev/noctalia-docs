If you wish to remove Noctalia from your system, you can uninstall the `noctalia` or `noctalia-git` package using your distribution's package manager.

### Manual Installation

If you installed manually using `just install release`, remove the files installed under your prefix:

```bash
rm ~/.local/bin/noctalia
rm -rf ~/.local/share/noctalia
```

## Cleaning Up

After removing the package, you may want to remove the configuration and cache files.

### Configuration and Cache

Noctalia stores its configuration and cache in the standard XDG directories. To remove them:

```bash
rm -rf ~/.local/state/noctalia
rm -rf ~/.cache/noctalia
```

### Generated Templates

If you enabled any templates in the **Color Scheme** settings, you will need to manually delete the generated files.
