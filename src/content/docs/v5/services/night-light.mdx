Applies color temperature shifts using the Wayland `wlr-gamma-control` protocol. Supported on compositors that implement gamma control (Hyprland, Sway, etc.). Follows the [`[location]`](/v5/services/location/) schedule when enabled; use `force` to ignore the schedule.

```toml
[nightlight]
enabled              = false
force                = false   # force night mode from startup

temperature_day   = 6500       # Kelvin (must be > temperature_night by at least 100)
temperature_night = 4000       # Kelvin
```

## IPC

```sh
noctalia msg nightlight-enable
noctalia msg nightlight-disable
noctalia msg nightlight-toggle
noctalia msg nightlight-force-toggle
```

`nightlight-enable` / `nightlight-disable` / `nightlight-toggle` control schedule enable state. `nightlight-force-toggle` toggles forced-on mode regardless of schedule.
