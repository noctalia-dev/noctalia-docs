Noctalia v5 is a new generation of the shell, with a native runtime, refreshed UI, and a configuration model built for the current desktop experience.

:::caution[Moving from v4?]
Noctalia v5 is a fresh install, not an automatic upgrade from the Quickshell-based v4 line. It ships as a separate package, your package manager will not replace v4 with v5, and v4 settings are not migrated.

Expect to recreate your compositor keybinds for opening the launcher, panels, and other Noctalia commands. The good part is that v5 ships with sane defaults, so you can get started quickly and customize from there.
:::

The v5 docs are still being built out. If you are setting up Noctalia for the first time, start with [Installation](/v5/getting-started/installation/), then [Running Noctalia](/v5/getting-started/running-the-shell/) for compositor startup and keybind setup. Once the shell is installed, [How configuration works](/v5/configuration/) explains where settings live and which file wins when the GUI and your config both set the same option.
